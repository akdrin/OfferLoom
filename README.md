# OfferLoom
OfferLoom is an AI-powered interview intelligence platform that analyzes a candidate’s resume against a job description and generates a structured, actionable interview preparation report.

Instead of generic feedback, OfferLoom produces a detailed evaluation including:

A match score (0–100) indicating profile alignment

Technical interview questions with:

The interviewer’s intention

A structured approach to answering

Behavioral interview questions with:

The reasoning behind the question

Guidance on framing strong responses

Identified skill gaps categorized by severity (low, medium, high)

A personalized, day-wise preparation plan with focused tasks

The system transforms unstructured inputs (resume + job description) into a structured interview readiness roadmap, helping candidates understand not just what to study — but why it matters.

Core Capabilities

Resume–JD semantic alignment scoring

AI-generated interview simulation questions

Skill gap analysis with severity tagging

Structured preparation planning

Scalable backend architecture with user-based evaluation storage

Tech Stack

Frontend: React

Backend: Node.js + Express

Database: MongoDB Atlas

Validation: Zod schema-based structured output

AI Integration: LLM-powered analysis engine

Vision

OfferLoom aims to move beyond resume optimization and into interview intelligence — helping candidates think like interviewers and prepare strategically rather than reactively.
